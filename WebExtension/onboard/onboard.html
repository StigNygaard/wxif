Congratulations!

xIFr version has been installed.

With this tool you can see a lot of the metadata that can be embedded in JPEG files. Various data of types EXIF, IPTC and XMP are supported.

New in xIFr version 2.1 are keyboard shortcuts in meta data popup window:

FunctionShortcut key
Close windowEscape
Show Image Data tabi
Show Map tabm
Show Larger Map (When on Map tab)l
Copy to Clipboardc
Open Options Pageo

To see (if there are) any recognized metadata in an image, you can usually just right-click the image and choose "View EXIF data" in your browser's contextmenu. To learn xIFr better to know, and especially to see what the "Deep Search" feature is all about, I have also created a little introduction or demonstration page:

www.rockland.dk/xIFr/start

Finally, make sure to visit xIFr's Options Page where you can adjust some aspects of how xIFr looks and behaves. Look for the wheel in the metadata window (or just press "o") to access the options.

I hope you'll find xIFr useful and enjoy using it...
